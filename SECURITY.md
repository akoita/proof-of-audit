# Security Policy

## Reporting a vulnerability

If you discover a security issue in Proof-of-Audit, please report it privately to the maintainer instead of opening a public issue.

Until a dedicated security contact channel is added, use the repository owner's GitHub account for responsible disclosure coordination.

## Scope note

This repository is currently a prototype. Do not rely on it for production fund flows, custody, or adversarial smart contract review without an additional security review.
